Project Title: Sports Photography Portfolio

Concept:
This project is a personal sports photography portfolio designed to showcase my work, projects, and professional identity as a photographer. The site emphasizes bold visuals, strong typography, and clean layout design to highlight the intensity and emotion of sports. Inspired by modern creative portfolios and sports media brands, the goal was to create a professional, visually engaging website that presents my best work while also demonstrating technical skills in web design, accessibility, and performance optimization.

Pages Included:

1. Main Page (index.html)
   The website is a single-page portfolio that includes multiple sections: a hero section with a background image and introduction, a featured work gallery displaying my top images, a projects section with horizontal scrolling image carousels, an about section describing my experience and goals, and a contact section with a form for user interaction. The page also includes a navigation bar and footer with social links.

Sketching and Planning Process:
The planning process began with hand-drawn sketches to determine layout, hierarchy, and user flow. I mapped out each section of the page, including the hero, gallery, projects, and contact areas. These sketches helped guide the structure of the HTML and the spacing system used in CSS. I focused on creating a clean, sports-inspired aesthetic that emphasizes imagery and readability.

Special Features and Techniques:

* Use of CSS custom properties (variables) for consistent design system
* Flexbox and CSS Grid for layout structure
* Masonry-style gallery using CSS columns
* Horizontal scrolling image carousels for project sections
* Responsive design using media queries for tablet and mobile devices
* Mobile navigation menu implemented using a checkbox (no JavaScript)
* Semantic HTML structure with proper sectioning elements
* Accessibility improvements using ARIA attributes and descriptive alt text
* SEO optimization including meta tags, Open Graph, and Twitter tags
* Image optimization using WebP format, lazy loading, and width/height attributes
* Performance improvements including preloading the hero image and optimized font loading
* CSS transitions and subtle animations for interactivity

Challenges and Solutions:
One challenge I encountered was maintaining consistency across different sections of the website, 
especially between the gallery and project sections. Since these sections display images in different 
layouts, it was important to ensure spacing, sizing, and alignment remained visually strong. I 
solved this by using CSS variables and a consistent spacing system throughout the site.

Another challenge was ensuring proper accessibility and SEO optimization. Simply because its hard to
pin point every piece of html that needs some sort of label. I also improved accessibility by adding 
descriptive alt text to all images and ensuring proper labeling of sections and forms.

Performance optimization was another key challenge, particularly due to the large number of high-quality 
images. I addressed this by compressing images, converting them to WebP format and adding lazy loading. 
I also optimized font loading by only including the weights used in the project.

Overall, these challenges helped improve both the visual quality and technical performance of the final 
website, resulting in a clean, responsive, and professional portfolio.
